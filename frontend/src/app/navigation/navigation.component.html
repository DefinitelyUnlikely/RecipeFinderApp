How to use this app:

Type your desired ingredient into the white box and press "Add ingredient" to add an ingredient to the list. Repeat for each desired ingredient. When you are ready to search for recipes press "Search".

How does it work?

The application will search for recieps containing all the ingredients added to the list. You'll be returned a list of recipes containing your selected ingredients.

Future releases will contain a strict search mode, where matching recipes MUST contain selected ingredients but nothing else.
